# Hangman Game

### What-
It's basically a word guessing game. User guess a character. When miss guess gradually lives ended at-least 6 lives.

### Pseudocode-
- create words list- to pick a randomly word
- show "-" how length words - it's help to user guess how long word
- get user input() with while loops
- if user input == correct list, print() this character
- if not match end lives, when lives == 6
- Game over
