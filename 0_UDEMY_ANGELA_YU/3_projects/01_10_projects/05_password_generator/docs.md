# Password Generator

### What-

This is password generator project. There are two step. One easy letter,digit,symbol [e.g., abcd123!@#]. Two hard is randomly generate, not flow serial.

### Pseudocode-

- import random- by use hard password generator
- input()- how many letter
- input()- how many digit
- input()- how many symbol
- user choose "easy" or "hard" option
- if easy
- password [abc123!@#]
- if hard
- password [#b2a!c3@] (random)
