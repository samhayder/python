# Tips Calculator

### What-

This project generate tips calculator. That means, user provide restaurant bill amount, tip (10%,12% or 15%) & how many people.

### Pseudocode-

- input()- bill amount $ to [float type]
- input()- tip amount %
- tip amount convert % [(bill * tip) / 100]
- input()- how many people to bear this bill [int type]
- tip result - [(bill + tip%) / people]
- result should print() .2 digit- use round()
