# Rock-Paper-Scissor Game

### What-
user choose a option & computer choose a option. Game rule-
Rock beat Scissor
Paper beat Rock
Scissor beat Paper

### Pseudocode-

- import radom to user random.choice()
- use while loop until the game win or lose or exit
- input()- choose a option
- print()- this option ASCII art
- computer - choose a option
- print()- this option ASCII art
- START if conditions to game rule
